Frequently Asked Questions

1. the difference between var, let, and const
Var is outdated and may result in bugs if we redeclared. instead, update the values. Const is rock solid for values we don't want to alter!
2. the difference between map(), forEach(), and filter()
Each item is array by map() to produce a new array. forEach() does not create a new array; it simply loops through items. Filter() creates a new array by selecting items that fit your rule.
3. explain arrow functions and how they are different from regular functions
Arrow functions (=>) are a shorter way to write functions.
4. how JavaScript Promises work
Promises are like IOUs for async tasks. They’re either pending, fulfilled (yay, data!), or rejected (oops, error).
5. how closures work in JavaScript
Closures let a function remember variables from its parent, even after the parent’s gone.
